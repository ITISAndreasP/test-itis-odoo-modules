Textblock

Einfügen von Texten in Angebote und Rechnungen

Textblöcke und -zeilen erstellen

Mit Hilfe dieses Moduls, können Textblöcke als Vorlagen abgelegt und in Belegen die Angeboten, Aufträgen oder Rechnungen eingefügt werden. Auch das Einfügen einzelner Zeilen ist möglich.

Video ansehen

Features

  • Textzeilen in Belegen
  • Textblöcke / Vorlagen erstellen und einfügen

Anwendungsbeispiele

Ein Nutzer kann beispielsweise einen Textblock für die Anrede erstellen.
Hierzu steht im Menü Verkauf, die Option Textblock zur Verfügung. In diesem Menü, können neue Einträge hinzugefügt oder bestehende Textblöcke bearbeitet werden.

Weiterhin kann beispielsweise über Produkte ein Textbaustein in den Artikeln eingefügt werden.
